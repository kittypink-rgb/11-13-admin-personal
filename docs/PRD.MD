# GESTOR DE PERSONAL

## **1. Descripción general del producto **
   - Nombre: Gestor de personal
   - Función: Organizar datos personales, administrativos, pagos, horarios y cambios en la base de datos de cada empleado
   - Tipo: página web
## **2. Objetivos **
   - Automatizar la gestión de empleados, nóminas, asistencia y administración interna de una empresa
   - Facilitar la organización de datos para el cargo de cada empleado, con el fin de evitar equivocaciones con el desempeño, asistencia y/o datos personales de cada usuario.
## **3.Tipos de usuarios en la plataforma **
   ### - **Administrador/jefe: **
          ► Acceso total al sistema 
          ► Generación de reportes 
          ► Acceso a la configuración de la plataforma
   ### - **Supervisor: **      
        ► Visualización del equipo
        ► Gestión de horarios, asistencias, entradas y salidas de cada empleado
        ► Aprobación de permisos y/o incapacidades
   ### - **Empleado: **
        ► Registro de asistencia
        ► Solicitud de permisos
        ► Consulta de horarios y/o novedades
        ► Visualización de historial.
## **4. Tipos de Gestión: **        
     - Pagos y nóminas
     - Asistencias y faltas
     - Historial laboral
     - Descuentos salariales por faltas

### Resumen: El producto sirve para controlar de una manera mas sencilla la parte administrativa de una empresa, para evitar riesgos de pérdidas, corte de personal mal ejecutado e incluso demandas por negligencia empresarial.
