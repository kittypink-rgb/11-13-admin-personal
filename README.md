         GESTOR DE PERSONAL             

Una plataforma web diseñada para organizar y gestionar
de forma eficiente la información de los empleados:
datos personales, horarios, asistencia y pagos.

Permite automatizar procesos administrativos,
reducir errores y mejorar la organización empresarial.

Incluye tres roles principales:
• Administrador → Control total y reportes
• Supervisor   → Gestión de equipo y asistencia
• Empleado     → Consultas y solicitudes

En resumen: una solución práctica que optimiza la
gestión interna y disminuye riesgos laborales.
